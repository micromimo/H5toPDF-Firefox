# H5toPDF-Firefox
「初步开发中」一个将HTML5网页转为PDF档案的Firefox扩展插件，并非调用Print机制，而是直接根据捕获的DOM转为PDF，保留所有样式和带格式文本，确保输出的PDF与您所见到的网页完全一致。并且支持按指定像素尺寸及比例输出，以适应多元需求。A Firefox WebExtension that converts HTML5 pages to PDF directly from the DOM. Bypass the print engine to preserve exact styles, with customizable pixel dimensions and aspect ratios.
